# react-cplayer

## Install

```bash
npm install --save react-cplayer
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'react-cplayer'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
